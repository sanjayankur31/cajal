Cajal The computational neuroscience laboratory map

Info bar

Information will appear here when someone clicks a marker on the map

This map attempts to show the different laboratories around the world that carry out computational neuroscience research. Click on an icon to view information about the laboratory in the information bar.

Based on OpenStreetMap and Bootstrap. Add your laboratory by opening a pull request on GitHub!
